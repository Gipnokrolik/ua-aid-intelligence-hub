Project Vision
UA Aid Intelligence Hub is an open-source data engineering initiative aimed at aggregating, processing, and analyzing donation data from the largest Ukrainian humanitarian and military funds (e.g., Come Back Alive, United24, Prytula Foundation).

The goal is to provide a transparent, data-driven overview of global support for Ukraine through advanced ETL pipelines and interactive BI dashboards.

Key Objectives
Multi-Source Aggregation: Building resilient scrapers for various platform APIs and web interfaces.

Data Standardization: Normalizing diverse data formats into a unified schema for cross-platform comparison.

Trend Analysis: Identifying seasonal patterns, donor geography, and platform-specific behavior.

Public Transparency: Providing reproducible tools for independent financial auditing of aid flows.

Tech Stack
Extraction: Python (Requests, Cloudscraper)

Storage: SQLite / PostgreSQL

Orchestration: Docker-compose

Analytics: SQL & Apache Superset
